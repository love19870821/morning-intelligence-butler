# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-04-14

This release focuses on the first stable public workflow:

- clear input validation for missing files and invalid JSON
- auto-created output directories
- explicit output format selection with legacy flag compatibility
- sample input generation and demo bundle creation
- module and console-script entry points
- first-run smoke test for local verification
- summary counts in rendered reports
- dynamic version metadata shared by source and packaging
- release-oriented ignore rules for build artifacts
